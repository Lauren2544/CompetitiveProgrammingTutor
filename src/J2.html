Loops
- Computers are often used to automate repetitive tasks, there are two different types of loops: for loops and while loops
- For loops are used when you know how many times the loop should run; it runs until it has run through as many iteration as there are items in the set
- While loops are used if you want the loop to break based on a condition other than the number of times it runs; it runs until some condition becomes false

While Loops
Flow of execution:
1. Evaluate the condition yielding True or False
2. If the condition is True the loop will execute the body and then go back to step one until the condition is false in which case the loop will be completed, it is important to note that if the variables in the loop are not incremented in the body of the loop this will create an infinite loop
- The break statements can also be used to stop the loop even if the condition is true
- Continue statements on the other hand can stop the current iteration, and continue with the next iteration
- For more information make sure to check out this link

For Loops
Flow of execution:
1. Checks if it is allowed to run through the first iteration
2. Runs through the first iteration
3. Checks if it is allowed to run through again if it is not the loop is complete
- The pass statement will skip over the for loop
- You can use nested loops where the ‚Äúinner loop‚Äù will be completely executed for each iteration of the ‚Äúouter loop‚Äù
- The range() function can also be used in for loops to set a range in which the for loop is allow to iterate to, E.g. for x in range(6): print (x, end=‚Äù ‚Äù) will print 0 1 2 3 4 5
