Functions

Function calls
- Function: a named sequence of statements that perform a computation; once you have defined a function you can then call on it by name and it can return data as a result, we have already seen one example of a function by the name type, E.g. print(type(32))
- Python provides a number of important functions that we can use without needing to provide the function definition
- max() returns the largest value in a list or the largest character in a string
- abs() returns the absolute value for a given input
- len() returns the amount of characters in a string or other sets of values
- int() returns the value as an integer
